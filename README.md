# aiida-inq
AiiDA plugin for the LLNL developed INQ code.

## Basic installation

```bash
git clone https://github.com/LLNL/aiida-inq.git
cd aiida-inq
pip install -e .
```

Future work: Register the package with pypi for simple `pip` installation.
